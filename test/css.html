CSS

Cascading Style Sheets (CSS) are a specific type of document to help format and alter the appearance of a HTML Website. It is used to define styles of text, sizes of tables and font, as well as colours and can be used to determine different styling across different sections of a webpage.

Originally HTML would be used to define this styles especially back with HTML 1, 2 and 3. Whereas HTML Version 4 introduced the use of CSS documents and help create a uniform look across the pages which made up the website. IT allowed tables, text or sections to be styled as one uniform style or be styled seperately.

CSS Version 1

CSS 1 - Introduced in 1996. The first Version of the CSS language which provided basic styling and structure. Limited in the abilty to style but its properties are still used in current and modern versions of CSS. (No Longer supported or Maintained by W3C)

CSS Version 2 and 2.1

CSS 2 - Introduced in 1998. The Second Version of the CSS language which provided basic styling and structure with additional new options. Position elements, dorection of Text and linking media were some of the options brought into CSS. Similar to Version 1 it hosted limited styling but its old and new properties are still used in current and modern versions of CSS. (No Longer supported or Maintained by W3C)

CSS 2.1 - Introduced in 2011. This version was a culmination of the specifications for CSS 1 and CSS 2. the standard fixed styling issues and provided more new options in regards to styling the Web pages. (The current Standard supported or Maintained by W3C)

CSS Version 3

CSS 3 - Currently in Development. The Third Version of the CSS language, which is currently being worked on, will host backward compatibility for all Previous CSS Versions. It will host a new way to how a specification will be organised via modules such as "Selectors" or "Animations", allowing developers to incorporate these new features which modern browsers can already host.

Advantages of CSS

Advantage of using CSS are;

  • Seperation of Content, style and layout
  • Flexible and Efficent
  • Easy to Maintain and Update
  • A single External CSS file can be applied to multiple documents
  • Reduction of Complexity resulting in Faster Page Loading
  • A huge array of capabilties for high quality presentation of webpages
  • Multiple Device and Viewport optimisation
