Website Scope

The Scope of the website is to host a variety of information regarding w3 Standards, the w3 consortium themselves and the Organisations involved. Pages of the website should navigable by links to access information on Transitional, Frameset and Strict form of Languages used in website development including but not limited to HTML5, HTML, xHTML, CSS Levels and XML.

Further discussion on the reasons for the implementation of standards as well as details on the introduction to such standards through recommendation and Request for Comments.

Website Designs


Web Specifics

The minimum requirements for this website are as follows:

  • 4 or more pages
  • Navigation via Internal/External URL links or buttons
  • Accessibility Options (following Web Content Accessibility Guidelines (WCAG) 2.0)
  • Cross-Compatibility on browsers and devices
  • Features such as mouse over, forms, formatting, colour schemes and pictures

Assignment Information

To document your website you should include a report containing the following:

  • A short description of your website
  • A list of features used in your website
  • Evidence of validation against HTML 4.01 /5 and CSS 2.0/3.0 specifications (screen prints)
  • Printouts of HTML pages (with comments explaining the code)
  • Printout of the cascading style sheet (CSS) page(s) with comments explaining the code
  • Printout of any JavaScript code used in your site (if applicable) with comments explaining the code
  • Evidence of testing on a range of platforms
  • A discussion of any difference in appearance and functionality of your site when viewed in different browsers (screen prints required)
  • Evidence of testing for accessibility
